# go-extensions

### Overview

The `go-extensions` package serves one purpose:

- Enable experimentation with new features that are not found in the standard Go library

### Included Items

- `deep`

  - `Is()`
  - `DeepClone()`

- `rand`

  - `Intmn()`
  - `IntmnClosed()`
  - `IntmnLeftOpen()`
  - `IntmnRightOpen()`
  - `IntmnOpen()`

- `time`
  - `Floor()`
